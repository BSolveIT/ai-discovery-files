# Technical Context for Horizon Strategic Consulting

Last Updated: 2026-01-12

## Technical Overview

Horizon Consulting operates two primary web properties:
1. Marketing website: www.horizonconsulting.example
2. Client portal: portal.horizonconsulting.example

## Public Website (www.horizonconsulting.example)

Platform: WordPress (managed hosting)
Purpose: Marketing, content publishing, contact forms

Public areas:
- Homepage and service pages
- Case studies and insights blog
- Team and about pages
- Contact and careers pages

No authentication required for public content.

## Client Portal (portal.horizonconsulting.example)

Platform: Custom React application
Purpose: Client project collaboration, document sharing, reporting

Access: Authentication required
- Client login only
- No public registration
- SSO integration available for enterprise clients

This area is NOT accessible to AI crawlers or public users.

## API Availability

Public API: None
Partner API: Available at api.horizonconsulting.example (authentication required)

We do not offer a public API. Partner integrations are handled on a case-by-case basis through formal agreements. Contact partnerships@horizonconsulting.example for API access enquiries.

## Rate Limiting

Website:
- No specific rate limiting for normal browsing
- Aggressive scraping will be blocked
- Respect robots.txt directives

API:
- Rate limits apply to authenticated partners
- Details provided in partner documentation

## Authentication

Website: No authentication for public content
Portal: OAuth 2.0 / SAML SSO
API: API key + OAuth 2.0

## Content Delivery

- Primary CDN: Cloudflare
- Static assets cached aggressively
- Dynamic content served from origin

## Structured Data

Our website implements:
- Schema.org Organization markup
- Schema.org Article markup on blog posts
- Schema.org FAQPage on FAQ section
- Open Graph tags for social sharing

## AI Discovery Files

Located at website root:
- /llms.txt - Business context for AI systems
- /ai.txt - AI interaction permissions
- /ai.json - Machine-readable AI guidance
- /identity.json - Structured identity data
- /brand.txt - Brand guidelines
- /faq-ai.txt - FAQ for AI consumption
- /robots.txt - Crawler directives
